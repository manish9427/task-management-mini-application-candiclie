## Features

- Create, edit, and delete tasks
- Assign task status: Pending, In-Progress, Completed
- Filter tasks by status
- Persistent local storage (JSON via LocalStorage)
- Instant UI updates (no page reloads)

## Tech Stack
- React (Hooks) & LocalStorage (JSON)
